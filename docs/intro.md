# AWS AI Practitioner — Notas de Estudio

Este sitio contiene mis notas personales para la certificación **AWS Certified AI Practitioner**, organizadas en tres secciones principales.

---

## Estructura del Examen

La certificación está compuesta por **5 dominios** con los siguientes pesos:

| Dominio | Tema                                  | Peso |
| ------- | ------------------------------------- | ---- |
| **1**   | Fundamentos de IA y ML                | 20%  |
| **2**   | Fundamentos de IA Generativa          | 24%  |
| **3**   | Aplicaciones de los Foundation Models | 28%  |
| **4**   | Directrices para la IA Responsable    | 14%  |
| **5**   | Seguridad, Cumplimiento y Gobernanza  | 14%  |

---

## Cómo usar esta guía

### 📚 Readiness Path

Notas completas de los 5 dominios del examen oficial. Cada dominio incluye:

- Conceptos teóricos con diagramas
- Servicios de AWS relevantes
- Quiz con preguntas de práctica
- Resumen al final

### 🎯 SkillBuilder

Profundización en temas específicos de AWS SkillBuilder, como parámetros de inferencia avanzados y ataques de seguridad en prompts.

### 📦 Extra

Notas complementarias sobre temas no cubiertos completamente en los dominios principales: métricas de ML, algoritmos, embeddings, bases de datos vectoriales, y más.
